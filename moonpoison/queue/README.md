Queue
==
### 선입후출 방식의 자료구조.
### 난이도 : ★★☆☆☆
### 이해도 : ★★★★★
미끄럼틀에 아이들이 여럿 내렸을때를 생각하면 편하다.
먼저 들어간 값이 가장 먼저나오며, 가장 먼저들어 간 값에 접근이 가능하다.
난이도와 이해도는 이전 챕터인 Stack과 동일하다.
