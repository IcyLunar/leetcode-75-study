# Sliding Window

### 슬라이딩 윈도우란?

고정 사이즈의 윈도우가 이동하면서 윈도우 내에 값들을 이용하여 문제를 해결하는 알고리즘을 말한다.  
보통 슬라이딩 윈도우라고 하면 고정된 크기의 윈도우를 말하는데, 가변 크기의 윈도우를 이용하는 문제도 있다. 가변 크기의 윈도우를 이용하는 방식은 투 포인터 방식과 매우 유사하다.

슬라이딩 윈도우 기법은 배열에서 연속적인 부분 집합을 찾을 때 유용하여 사용할 수 있다.
