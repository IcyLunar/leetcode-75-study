# Two Pointers

### 투 포인터 알고리즘이란?

투 포인터 알고리즘은  배열에서 각각 다른 원소를 가리키고 있는 2개의 포인터를 조작해가면서 원하는 답을 찾을 때 까지 탐색을 진행하는 알고리즘이다.

보통 두 포인터를 `left, right 포인터` 혹은 `start, end 포인터` 라고 부른다.
