# Hash Map / Set

### Map 자료구조

- key와 value가 한 쌍을 이루는 집합
- key는 중복되지 않는다.
- 순서가 보장되지 않는다.

### Set 자료구조

- map과 달리 한 요소의 집합
- 중복을 허용하지 않는다.
- 순서가 보장되지 않는다.

### Map과 Set의 장점

- 탐색 속도가 빠르다.
- 중복값을 다루는 문제에서 사용할 수 있다.
