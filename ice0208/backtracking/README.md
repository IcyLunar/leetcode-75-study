# Backtracking

백트래킹은 재귀적으로 문제를 하나씩 풀어가면서 현재 재귀를 통해 확징 중인 노드가 제한된 조건에 위배되는지 판단하고, 만약 제한된 조건을 위배된다면 그 노드를 제외하고 다음 단계로 나아가는 방법이다.

DFS를 통해 모든 노드를 탐색하면서 더 이상 필요가 없는 부분을 쳐내는 행위를 백트래킹이라고 생각하면 된다.
