# Stack

### Stack 이란?

스택은 가장 마지막에 삽입된 자료가 가장 먼저 삭제되는 후입 선출 (Last In First Out) 구조의 자료구조이다.

### Stack의 예시

- 웹 브라우저에서 뒤로 가기
- 실행 취소 (Undo)
- 후위 표기법 계산
